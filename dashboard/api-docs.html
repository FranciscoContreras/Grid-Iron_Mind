High-performance NFL data API with AI-powered insights

Overview

Grid Iron Mind is a comprehensive NFL data API providing real-time and historical statistics, player information, team data, game schedules, injury reports, and AI-powered insights.

⚡ High Performance

Sub-200ms response times with aggressive Redis caching

🤖 AI-Powered

Claude 3.5 Sonnet and Grok with automatic fallback

🔄 Auto-Fetch

Automatically retrieves missing data on-demand

📊 Comprehensive

Complete 2025-2026 season (272 games, all 18 weeks)

Key Features

  • Players: Complete player profiles, stats, career history, injuries
  • Teams: All 32 NFL teams with rosters, stats, schedules
  • Games: Live scores, schedules, weather conditions, results
  • Statistics: Player and team stats, leaders, historical data
  • Defensive Analytics: Matchup analysis, rankings, player vs defense
  • AI Predictions: Game outcomes, player performance, insights
  • AI Data Garden: Self-healing database with natural language queries
  • Weather Data: Current, historical, and forecast weather

Authentication

Public Endpoints

Most endpoints are publicly accessible without authentication:

  • Players, Teams, Games
  • Statistics, Weather
  • Defensive stats, Injuries, Career stats

Protected Endpoints (API Key Required)

AI-powered endpoints require authentication via API key.

⚠️ API Key Required All endpoints under /api/v1/ai/* and /api/v1/garden/* (except health check) require an API key.

Players Endpoints

GET /api/v1/players

Get a paginated list of NFL players with optional filters.

Query Parameters

Parameter Type Description
limit integer Results per page (default: 50, max: 100)
offset integer Pagination offset (default: 0)
position string Filter by position (QB, RB, WR, TE, etc.)
status string Filter by status (active, injured, inactive)
team uuid Filter by team ID

Example Request

GET /api/v1/players?position=QB&limit=20&status=active
GET /api/v1/players/:id

Get detailed information for a single player.

GET /api/v1/players/:id/career

Get career statistics and team history for a player.

GET /api/v1/players/:id/injuries

Get injury reports for a specific player.

GET /api/v1/players/:playerId/vs-defense/:teamId New

Get historical performance data for a player against a specific defense.

Query Parameters

Parameter Type Description
season integer Filter to specific season (optional)
limit integer Number of games (default: 5, max: 50)

Auto-Fetch System

The API includes intelligent auto-fetching that automatically retrieves missing data when queried.

How It Works

  1. You query an endpoint (e.g., /api/v1/games?season=2025&week=1)
  2. If the database returns empty results, the system automatically fetches data from ESPN API
  3. Data is stored in the database and returned with X-Auto-Fetched: true header

Supported Auto-Fetch

  • Games: Automatically fetches game schedules
  • Players: Fetches rosters when individual player missing
  • Stats: Fetches game statistics from NFLverse
  • Teams: Ensures all 32 teams exist
⚡ Performance Note The first request may take 2-10 seconds while data is fetched, but subsequent requests are served instantly from the database.
