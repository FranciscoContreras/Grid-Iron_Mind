üöÄ Quick Start

Get started with your first API request in seconds:

Example: Get All Teams

curl https://nfl.wearemachina.com/api/v2/teams

Example: Get Player Stats

curl https://nfl.wearemachina.com/api/v2/stats/leaders?category=passing_yards&season=2025&limit=10

Response Format

All successful responses follow this format:

{
  "data": [...],
  "meta": {
    "total": 100,
    "limit": 50,
    "offset": 0,
    "timestamp": "2025-01-15T12:00:00Z"
  }
}

üîê Authentication

Most endpoints are public and don't require authentication. AI-powered endpoints require an API key.

Public Endpoints

No authentication required. Subject to standard rate limits (100 requests/minute).

AI Endpoints

Require API key authentication. Subject to strict rate limits (10 requests/minute).

curl -H "X-API-Key: your_api_key_here" \
  https://nfl.wearemachina.com/api/v2/ai/predict/game/[game-id]

Contact the API administrator to request an API key for AI endpoints.

üë§ Players

GET /api/v2/players

List all NFL players with optional filtering.

Query Parameters

Parameter Type Description
team UUID Filter by team ID
position string Filter by position (QB, RB, WR, etc.)
status string Filter by status (active, inactive, injured)
limit integer Results per page (default: 50, max: 100)
offset integer Pagination offset (default: 0)
GET /api/v2/players/:id

Get detailed information for a specific player.

Path Parameters

Parameter Type Description
id UUID Player ID
GET /api/v2/players/:id/career

Get career statistics for a player across all seasons.

GET /api/v2/players/:id/injuries

Get injury history and current status for a player.

GET /api/v2/players/:id/advanced-stats

Get advanced statistics including Next Gen Stats for a player.

Query Parameters

Parameter Type Description
season integer Filter by season year
week integer Filter by week number
stat_type string Filter by type (passing, rushing, receiving)

‚ö†Ô∏è Error Handling

All errors follow a consistent format:

{
  "error": {
    "code": "NOT_FOUND",
    "message": "Player not found",
    "status": 404
  }
}

Common Error Codes

Status Code Description
400 BAD_REQUEST Invalid request parameters
401 UNAUTHORIZED Missing or invalid API key
404 NOT_FOUND Resource not found
429 RATE_LIMIT_EXCEEDED Too many requests
500 INTERNAL_ERROR Server error

‚è±Ô∏è Rate Limits

Public Endpoints

100 requests/minute

Applies to all non-AI endpoints

AI Endpoints

10 requests/minute

Requires API key authentication

Rate limit headers are included in all responses: X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset

üìÑ Pagination

List endpoints support pagination using limit and offset parameters.

Parameters

Parameter Default Max Description
limit 50 100 Number of results per page
offset 0 - Number of results to skip

Example

GET /api/v2/players?limit=25&offset=50

Response

{
  "data": [...],
  "meta": {
    "total": 2515,
    "limit": 25,
    "offset": 50,
    "timestamp": "2025-01-15T12:00:00Z"
  }
